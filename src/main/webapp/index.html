Filmays

Cada vez hay más accesibilidad para ver películas, mediante servicios online que permiten verlas desde cualquier sitio ,mediante alguna suscripción, o incluso alquilar películas online por poco dinero.

Por esta razón queremos desarrollar Filmays, una aplicación que permita buscar información, valoración, trailer y merchandising de alguna película concreta. Pudiendo además crear lista de peliculas pendientes por ver, lista de peliculas favoritas, etc.
